## 📌 使用方式

### sing-box

在规则集配置中填写对应规则的 `.srs` 链接：

```json
{
  "type": "remote",
  "tag": "规则名称",
  "url": "https://raw.githubusercontent.com/milangree/rules/main/rules/singbox/规则名称/规则名称.srs",
  "format": "binary"
}
```

> 国内可使用 jsDelivr CDN 加速访问：
> `https://cdn.jsdelivr.net/gh/milangree/rules@main/rules/singbox/规则名称/规则名称.srs`

### mihomo

域名规则（`domain`）与 IP 规则（`ipcidr`）需分开引用：

```yaml
rule-providers:
  规则名称_domain:
    type: http
    behavior: domain
    format: mrs
    url: "https://raw.githubusercontent.com/milangree/rules/main/rules/mihomo/规则名称/规则名称_domain.mrs"
    interval: 86400
  规则名称_ipcidr:
    type: http
    behavior: ipcidr
    format: mrs
    url: "https://raw.githubusercontent.com/milangree/rules/main/rules/mihomo/规则名称/规则名称_ipcidr.mrs"
    interval: 86400
```
